学习路径

三个阶段的成长之旅

每个阶段都有明确的目标和可检验的里程碑，让学习不再迷茫。

Phase 01 🐣

小白入门

从安装 Python 开始，掌握基础语法、数据类型、流程控制。建立编程思维，写出第一个完整程序。

变量与类型 条件循环 函数定义 文件操作 异常处理
Phase 02 🐍

进阶提升

深入面向对象编程、模块化设计、常用库生态。能够独立完成中等规模的项目开发。

OOP 编程 迭代器生成器 装饰器 正则表达式 数据库操作 Web 开发
Phase 03 🐉

精通之路

掌握并发编程、性能优化、设计模式、源码阅读。具备架构设计能力，成为真正的 Python 高手。

异步编程 元编程 C 扩展 设计模式 性能调优 源码剖析
知识体系

核心知识点全覆盖

从基础到高级，构建完整的 Python 知识图谱。

📝 入门

基础语法

变量、运算符、字符串、列表、字典、元组、集合 —— 掌握 Python 的核心数据结构与基本运算。

🔀 入门

流程控制

if/elif/else 条件判断、for/while 循环、break/continue、列表推导式 —— 让程序拥有逻辑。

⚙️ 入门

函数与模块

函数定义与调用、参数类型、返回值、lambda 表达式、模块导入与包管理。

📁 入门

文件与 IO

文件读写、上下文管理器（with 语句）、JSON/CSV 处理、路径操作与目录遍历。

🏗️ 进阶

面向对象编程

类与对象、继承与多态、封装、魔术方法、属性装饰器、抽象基类 —— 构建清晰的代码架构。

🎯 进阶

高级函数特性

闭包、装饰器、迭代器、生成器（yield）、functools、偏函数 —— 写出优雅的 Pythonic 代码。

🌐 进阶

Web 开发入门

Flask / FastAPI 框架、RESTful API 设计、请求与响应处理、模板引擎、数据库 ORM。

🗄️ 进阶

数据库与存储

SQLite / PostgreSQL 操作、SQLAlchemy ORM、Redis 缓存、数据持久化最佳实践。

⚡ 精通

并发与异步编程

多线程（threading）、多进程（multiprocessing）、asyncio 异步 IO、协程、GIL 深入理解。

🔮 精通

元编程

type() 动态创建类、metaclass 元类、描述符协议、__init_subclass__、代码即数据。

📊 精通

性能优化

内存管理、对象池、cProfile 性能剖析、Cython 加速、JIT 编译（PyPy）、算法复杂度分析。

🔬 精通

CPython 源码

对象模型、内存分配机制、字节码执行、GIL 实现、C 扩展开发 —— 深入语言底层。
